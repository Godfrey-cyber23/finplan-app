## ✨ Features

### 🔐 Bank-Level Security
- **4-digit PIN protection** with visual feedback
- **Biometric authentication** support (fingerprint/face ID)
- **AES-256 encryption** for all financial data
- **Auto-lock** after inactivity (5 minutes)
- Data never leaves your device - **100% private**

### 💰 Complete Financial Management
- **Multi-account support** (checking, savings, credit cards)
- **Income source tracking** with multiple frequencies (monthly, weekly, bi-weekly, yearly)
- **Expense categorization** with 8+ categories
- **Transaction history** with search and filtering
- **Real-time balance updates**

### 🎯 Goal-Based Savings
- Set **custom savings goals** with target amounts and dates
- **Visual thermometer** progress indicator
- **Monthly contribution suggestions** based on your timeline
- **Goal completion celebrations** with confetti effects
- **8 color themes** for goals (amber, blue, green, purple, red, teal, pink, orange)
- **20+ emoji icons** to personalize your goals

### 📱 Modern Interface
- **Dark/Light mode** with system preference detection
- **3 color themes**: Professional (blue), Vibrant (cyan), Pastel (purple)
- **Custom accent color picker** - choose any color!
- **High contrast mode** for accessibility
- **Responsive design** works on mobile, tablet, and desktop
- **Smooth animations** and haptic feedback

### 🔔 Smart Notifications
- **Budget threshold alerts** (80% and 95% of monthly budget)
- **Large transaction detection** (>$200)
- **Weekly spending summaries**
- **Browser push notifications** support
- **In-app notification center** with history

### 📈 Intelligent Insights
- **Financial Health Score** (0-100) based on:
  - Savings rate
  - Spending trends
  - Budget utilization
- **Spending trend analysis** month-over-month
- **Category breakdown** with percentage visualization
- **Automated money-saving tips**
- **Goal feasibility analysis**
- **Large expense detection**

### 🧾 Smart Receipt Scanner
- **Camera capture** or **photo upload**
- **OCR text recognition** (powered by Tesseract.js)
- **Automatic amount detection** from receipt images
- **Automatic description extraction**
- **Receipt storage** in IndexedDB
- **Receipt viewer** for later reference

### ✅ Task Management
- Built-in **task planner**
- **Due date tracking**
- **Completion status filtering**
- **Quick add** functionality

### 📊 Comprehensive Reporting
- **Monthly summary reports**
- **Spending by category** with percentages
- **Month-over-month comparisons**
- **Key metrics dashboard** (income, expenses, savings)
- **Transaction history** with receipt indicators

### 🎨 Customization
- **6 navigation tabs**: Home, Income, Budget, Goals, Bank, Tasks
- **Balance visibility toggle** (hide/show amounts)
- **Filterable transaction lists**
- **Sortable transaction tables**
